# User Stories: Auraskill Application

---

## 1. User Authentication

*   **US001:** As a user, I want to securely log in to the Auraskill system using Keycloak so that I can access my professional profile and features.

## 2. Professional Profile Management

*   **US002:** As an authenticated user without a professional profile, I want to create my professional profile so that I can build and manage my professional identity within the Auraskill system.
*   **US003:** As an authenticated user, I want to view my personalized professional dashboard with my key profile information (name, photo, biography, top education, experience, and awards) so that I can quickly review my professional identity and access relevant features.
*   **US008:** As an authenticated user, I want to edit and update my professional profile information so that I can keep my professional identity current and accurate.
*   **US009:** As an authenticated user, I want to permanently delete my professional profile so that I can remove my data from the system.
*   **US010:** As an authenticated user, I want to add, edit, and delete my professional experience entries so that I can accurately reflect my career history.
*   **US011:** As an authenticated user, I want to add, edit, and delete my professional education and certification entries so que eu possa refletir com precisão minhas realizações e qualificações acadêmicas.
*   **US013:** As an authenticated user, I want to add, edit, and delete my professional skills and indicate my proficiency level so that I can accurately represent my capabilities.
*   **US014:** As an authenticated user, I want to add, edit, and delete my professional awards and recognition entries so that I can showcase my achievements.
*   **US015:** As an authenticated user, I want to add, edit, and delete my professional project entries so that I can highlight my key accomplishments and contributions.
*   **US016:** As an authenticated user, I want to add, edit, and delete my professional contact information (e.g., email, phone, social links) so that I can control how recruiters and others can reach me.
*   **US017:** As an authenticated user, I want to add, edit, and delete my professional social media and portfolio links so that I can showcase my online presence and work.
*   **US018:** As an authenticated user, I want to update and maintain my professional biography so that I can accurately describe my professional narrative.
*   **US019:** As an authenticated user, I want to upload, change, or remove my professional profile picture so that I can personalize my profile's visual representation.

## 3. Career Data Centralization (Existing from Business Use Cases)

*   **US004:** As a professional, I want to centralize all my experiences and skills in one place so that the system stores this data as my "Single Source of Truth".

## 4. Segmented Resume Generation (Existing from Business Use Cases)

*   **US005:** As a professional, I want to generate segmented resumes (e.g., for a specific job role) by selecting only relevant projects and certifications so that I can tailor my application for different opportunities.

## 5. Prime Profile Acquisition (Existing from Business Use Cases)

*   **US006:** As a Standard User, I want to acquire a Prime Profile so that I can access exclusive templates and advanced features.

## 6. Resume Generation

*   **US007:** As an authenticated user with a complete professional profile, I want to generate a tailored professional resume in PDF format, optimized for a specific job opportunity, so that I can increase my chances of being noticed by recruiters.

## 7. Access Control

*   **US012:** As an authenticated user, I want to see and access only the menu items and features relevant to my assigned user profile (e.g., Common, Premium, Business, Administrator) so that I have a personalized and secure experience.
